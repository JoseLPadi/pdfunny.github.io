Política de Privacidad

Bienvenido a Light Sudoku. Tu privacidad es importante para nosotros.

1. Datos personales

No recopilamos datos personales. El usuario se autentica de forma anónima utilizando Firebase Authentication.

2. Uso de datos

Se genera un identificador anónimo únicamente para permitir el funcionamiento del juego. No se asocia a información personal.

3. Publicidad y Proveedores Terceros

Utilizamos Appodeal para mostrar anuncios. Para ello, se pueden recopilar identificadores técnicos como la dirección IP y el ID de publicidad del dispositivo. Consulta la Política de Privacidad de Appodeal para más información.

4. Compartición de datos

No compartimos ni vendemos datos personales a terceros, fuera de los identificadores técnicos necesarios para la publicidad mencionada anteriormente.

5. Cambios

Esta política puede actualizarse en futuras versiones.
